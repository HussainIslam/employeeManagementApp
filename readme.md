Employee Management System
==========================
I am creating a employee management system for companies. Till this far, I have used the following technologies to build this app:
  * Node.js
  * Vanilla Javascript
  * Express.js
  * MongoDB
  * Bootstrap
  * Ajax
  * JSON
  * RESTful API
  * Mongoose.js
  * Express-handlebars
  * Body-parser
  * Multer
